A themeable progress bar library for everything.

ProgressJs is a JavaScript and CSS3 library which helps developers to create and manage a progress bar for any object on the page. You can design your own template for progress bar or customize it simply.

You can use ProgressJs to show the progress of loading content (images, videos etc.) on the page to the users. It can be used on all elements including textboxes, textareas or even the whole body.

It is light-weight, easy to use, customizable, free and open-source.
